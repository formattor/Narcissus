# 快捷键

## VS Code

|快捷键|功能|
|--|--|
|`shift + alt + 鼠标左键`|跨行同列选中|
|`ctrl + D`|添加下一个同名内容|
|`Ctrl + enter` | 行中换行 |
|`alt + ↑`| 代码行向上移动 |
|`ctrl + l`|选单行|
|`alt + cursor`|多选光标所在位置|
|`ctrl + 0`|定位文件|
